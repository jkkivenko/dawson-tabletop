# Description

A large ion-powered engine assembly. Like all Ion Engines, its Speed suffers within an atmosphere.

# Effects

The Spacecraft's Engines become the following:

| Power Draw | Speed (Vacuum) | Speed (Atmosphere) |
| -----------|----------------|--------------------|
| 1 | `32 m/s` | `8 m/s` |
| 2 | `46 m/s` | `12 m/s` |
| 3 | `60 m/s` | `15 m/s` |
| 4 | `74 m/s` | `19 m/s` |
| 5 | `88 m/s` | `22 m/s` |
